== Description ==
Allows access to Ushahidi's verified reports through Open311's GeoReport v2 system

== Installation ==
1. Copy the entire /open311/ directory into your Ushahidi instance's /plugins/ directory.
2. In the Admin screen, navigate to the Addons tab and activate the plugin.
